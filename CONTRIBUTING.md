# Contributing

Research Judge is meant to stay small, readable, and easy to adapt.

Good contributions:

- improve the workflow without making it bloated
- add focused examples
- make evidence grading clearer
- reduce overconfident or unsupported behavior
- improve compatibility with agent skill systems

Avoid:

- persona-heavy language
- claims that the skill makes an agent “expert-level”
- instructions that override user, system, safety, or organization policies
- broad frameworks that make the skill harder to use
